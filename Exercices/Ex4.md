
# Exercice 4

Créer une API pour une gestion de rendez-vous. Les utilisateurs peuvent ajouter, mettre à jour, récupérer et supprimer des rendez-vous à l'aide des méthodes HTTP GET, POST, PUT et DELETE.

On utilisera le no sql

Un rendez-vous est composé de : 

- titre string required, 
- description string, 
- date_debut : date, 
- date_fin : date, 
- done : boolean
