# Exercice 1

L'objectif de cet exercice est de se familiariser avec l'utilisation de Express JS.

1. Créer un projet Express
2. Créer un serveur Express qui peut écouter sur le port 3000
3. Créer une route helloworld qui retourne "Hello World" sur /helloworld
4. Créer une route /params/:name qui retourne "Le paramètre est :name" en remplacant :name par le paramètre
5. Créer une seconde route /query?name=:name qui retourne "Le paramètre est :name" en remplacant :name par le paramètre
6. Créer une route /not-found qui retourne "Not Found" avec un code 404
